# Data-Driven Analysis of Tree Suitability for Urban Roadsides (Internship)
Industrial training project analyzing urban tree suitability in Seberang Perai, Malaysia using Python, Excel, and Power BI dashboards, applying MCDA, evaluating supervised models, and developing EcoSphere AI system for prediction and decision support.

```
Features / Key Activities

Data cleaning, preprocessing, and validation using Python and Excel
Multi-Criteria Decision Analysis (MCDA) for tree suitability
Evaluation of supervised machine learning models
Interactive dashboards in Power BI
EcoSphere AI system for decision support and future prediction
```
4. Tools & Technologies

Python (pandas, scikit-learn, etc.)
Microsoft Excel
Power BI
QGIS (for geospatial validation)
